# MangoX — Plataforma patrimonial

Stack: React 18 + TypeScript + Vite + Tailwind CSS 4 + Supabase + i18n.

## Scripts
- dev: entorno de desarrollo
- build: compilación para producción
- lint: reglas de calidad
- preview: vista previa local

## Portabilidad
- Exportar/Importar datos: se definirá en siguientes versiones.
- Despliegue: compatible con Vercel (ver vercel.json).

## Notas
- Detección automática de idioma (ES/EN) y tema del sistema.
- UI base estilo Apple con glassmorphism y FAB multiacción.
